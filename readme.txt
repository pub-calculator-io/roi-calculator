== Description ==

Free return on investment (ROI) calculator that helps investors calculate return on investment, investment gain, and annualized ROI rates. 

== Installation ==

1. Upload the /ci_roi_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the [Return on Investment (ROI) Calculator](https://www.calculator.io/roi-calculator/ "Return on Investment (ROI) Calculator Homepage") plugin through the "Plugins" menu in WordPress.
3. Configure your settings via Appearance > Widgets > Return on Investment (ROI) Calculator.
4. And you're good to go!

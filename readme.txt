Free return on investment (ROI) calculator that helps investors calculate return on investment, investment gain, and annualized ROI rates. 

[https://www.calculator.io/roi-calculator/](https://www.calculator.io/roi-calculator/)

== Usage ==

Add the Return on Investment (ROI) Calculator shortcode to your page, post or sidebar:

`[ci_roi_calculator]`

Add the following code to your template where you would like the Return on Investment (ROI) Calculator to appear:

`<?php display_ci_roi_calculator(); ?>`

== Installation ==

1. Upload the Return on Investment (ROI) Calculator /ci_roi_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Return on Investment (ROI) Calculator plugin through the "Plugins" menu in WordPress.
